## Short Description Draft

Secure fleet, lending, and credit workflow operations in one mobile workspace.

## Full Description Draft

FMS Mobile helps authorized teams manage fleet operations, borrower records,
loan workflows, supporting documents, and audit-ready credit processes from a
single secure mobile workspace.

Core capabilities in the current product include:

- role-based sign-in and secure account access
- borrower and loan application workflow management
- supporting document handling and AI-assisted document parsing
- meeting-audio transcription and meeting-minutes workflows for authorized users
- credit, risk, and operational dashboards for internal teams
- audit-conscious workflow visibility and account-management features

This app is intended for authorized operational and credit-workflow use cases.
Users should only submit information they are authorized to collect, review,
and manage.
